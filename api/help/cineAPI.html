For Cinema

Please use this API as /api/cinema/...

endpoint description example
GET / Retrieve all cinema GET /
GET /:id Retrieve single cinema by id GET /1
DELETE /:id Delete single cinema by id DELETE /1
POST / Create new cinema
title, director, year, actors(string with commas),
country fields are required		 POST /

For Users

Please use this API as /api/user/...

endpoint description example
GET / Retrieve all users GET /
GET /:id Retrieve single user by id GET /1

For Authentication

Please use this API as /api/auth/...

endpoint description example
POST /signin Sign in
id, password fields are required		 POST /signin
POST /register Create new user
username, email, password fields are required		 POST /register
DELETE /:id Delete single user by id DELETE /
POST /signedstatus Check if a client is signed in POST /signedstatus
